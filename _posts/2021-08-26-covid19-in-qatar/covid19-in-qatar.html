Data Description

The data is sourced from the tidycovid19 R package created by Joachim Gassen on Github, which in turn sourced the data from The Johns Hopkins University Center for Systems Science and Engineering (JHU CSSE) publicly available Github repository. JHU CSSE collated the numbers from the responsible body of each country.

The data contains the following variables:

variable class description
country character Country name
iso3c character ISO3c country code as defined by ISO 3166-1 alpha-3
date date Calendar date
daily_confirmed numeric Daily increase in Covid-19 cases
acc_confirmed numeric Accumulated Covid-19 cases
daily_deaths numeric Daily increase in Covid-19 deaths
acc_deaths numeric Accumulated Covid-19 deaths
daily_recovered numeric Daily increase in Covid-19 recoveries
acc_recovered numeric Accumulated Covid-19 recoveries

What can be noticed from the table is the extremely low number of total deaths at 601 from a total of 227,000 cases, representing a death rate of only 0.26%, which is extremely low compared to the global average of 2% (World Health Organisation [WHO], 2021). Several factors contributed to this figure. One of which is that Qatar has a relatively low median age at 32.3 years (Worldometer, 2021). Additionally, a fine of up to $53,000 has been instated for anyone who does not wear a face covering in public, and as per a Qatari official “anyone who tests positive for coronavirus immediately receives high-quality medical treatment at no cost” (Tadros, 2020).
